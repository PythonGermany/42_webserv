Request processing flow (incomplete): -----------------------------------------
1. Parse request line
  - if not valid, return 400
2. Parse headers
  - if not valid, return 400
3. Check for Host header
  - if not present, return 400
4. Check version
  - if not valid, return 505
5. Find correct VirtualHost
  - if not found, return 500
6. Find correct location context
  - if not found, return 500
7. Check if method is allowed
  - if not, return 405
8. Check if path exists
  - if not, return 404
9. Check if request method requires to wait for body
  - if so, check if content-length is present
    - if not, return 411
10. Check if location is redirecting
  - if so, return 301
11. Check if path exists
  - if not, return 404
12. Check if path is a directory
  - if so, check if path ends with "/"
    - if not, return 301 "path/"
    - if so, check if index file exists
      - if so, return 200
      - if not check if directory listing is allowed
        - if so, return 200
        - if not, return 403
13. Check if path is readable
  - if not, return 403
14. Load file
  - for error, return 500
  - for success, return 200
